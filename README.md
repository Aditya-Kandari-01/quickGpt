quickGpt

  Quickgpt is a full-stack web application that provides a ChatGPT-style interface, allowing users to    interact with AI-powered conversational models in real time. It features a modern frontend and a       lightweight backend API to process and respond to user inputs.

Features
  Real-time Chat: Send and receive messages instantly.
  AI-powered Responses: Integrates with AI models to generate human-like replies.
  User-friendly Interface: Clean and responsive UI built with React.
  Theme Support: Light and dark mode support.
  History Management: View past conversations in a sidebar.
  Tech Stack
  Frontend: React, Tailwind CSS (or your CSS framework)
  Backend: Node.js, Express
  API: Connects with OpenAI or similar AI services for responses
  Utilities: Axios for HTTP requests, React Hot Toast for notifications
Installation
  Clone the repository
    git clone https://github.com/Aditya-Kandari-01/quickGpt.git
    cd quickGpt
  Setup Backend
    cd backend
    npm install
    npm start
  Setup Frontend
    cd ../client
    npm install
    npm start
    Open your browser at http://localhost:3000 to access the app.
  Usage
    Type your message in the input box.
    Press enter or click the send button to submit.
    View AI-generated replies in the chat window.
    Switch between light and dark themes using the theme toggle.
